Instrucciones:
- Coloca audio1.mp3 y audio2.mp3 en la carpeta assets/
- Abre index.html para empezar el juego
- game.html: selecciona las opciones de cada filtro
- feliz.html: muestra mensaje de cumple
- secreto.html: carga los audios
- Estilo: gris oscuro de fondo y rojo apagado
- Compatible con cualquier navegador moderno
